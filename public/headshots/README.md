# Headshots

Drop the three tutor headshots in this folder, then update the `photo` paths in
`lib/content.ts` to match the filenames.

Defaults expected by the site:

- `tutor-one.jpg`
- `tutor-two.jpg`
- `tutor-three.jpg`

Tips:

- Square images look best (they're displayed in a square crop). ~600×600px is plenty.
- Until a real image is added, the site shows a friendly initials placeholder.
